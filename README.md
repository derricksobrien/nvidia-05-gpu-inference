# GPU-Accelerated AI Inference

> **Track:** NVIDIA HPC & Developer Upskilling (Track 2)
> **Course 5 of 8**

Deploy low-latency model inference at scale using TensorRT, quantization, and the Triton Inference Server.

---

## Modules

### Module 1: TensorRT Fundamentals

**Learning Objectives:**
- Convert ONNX models to TensorRT engines
- Apply layer fusion optimizations
- Validate TensorRT vs. PyTorch output accuracy

### Module 2: Model Quantization

**Learning Objectives:**
- Apply INT8 and FP16 quantization
- Calibrate quantization with representative data
- Measure accuracy-performance tradeoffs

### Module 3: Triton Inference Server Setup

**Learning Objectives:**
- Deploy models to Triton
- Configure batching policies
- Test concurrent inference throughput

### Module 4: Throughput & Latency Benchmarking

**Learning Objectives:**
- Measure P50/P95/P99 latency
- Identify GPU memory bandwidth bottlenecks
- Achieve sub-500ms inference targets
